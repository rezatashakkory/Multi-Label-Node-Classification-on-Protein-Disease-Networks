# Multi-Label-Node-Classification-on-Protein-Disease-Networks
Disease-gene association prediction using BioGRID protein network (19,765 genes, 305 diseases). Discovered low homophily (2.5%) drives Label Propagation's superiority over GNNs. Hybrid ensemble: 60% LP + 20% SIGN + 20% XGBoost (AP: 0.057353). Key: match methods to data properties.
